# Sistema de Controle Policial - API com Clean Architecture

Este é o repositório oficial da API responsável pelo gerenciamento do sistema de controle policial. Utilizamos os princípios da Clean Architecture para estruturar e desenvolver esta aplicação.

## Desafio e Objetivo

O desenvolvimento desta API tem sido uma jornada desafiadora e extremamente gratificante. Meu objetivo principal é criar um sistema robusto e seguro para gerenciar dados e informações cruciais sobre indivíduos associados a atividades criminosas.

## Principais Recursos

- **Gestão de Bandidos:** Permitimos o registro, atualização e consulta de informações detalhadas sobre indivíduos ligados a atividades criminosas.
- **Controle de Dados:** Garantimos a integridade e segurança dos dados, implementando estratégias de acesso e manipulação seguras.
- **Arquitetura Limpa:** Utilizamos a Clean Architecture para garantir a separação clara de responsabilidades entre os componentes do sistema, facilitando a manutenção e evolução da aplicação.

## Tecnologias Utilizadas

- **Linguagem:** .NET8 (C#)
- **Frameworks e Bibliotecas:** 
  - ASP.NET Core: Utilizado para construir a API web.
  - Entity Framework (EF): ORM (Object-Relational Mapping) para manipulação de banco de dados.
- **Bancos de Dados:** MySQL

## Instalação e Uso

Para utilizar a API, siga os passos abaixo:

1. Clone o repositório: `git clone https://github.com/FrancoEdu/pmespAPI.git`.
2. Atualize o banco com migrations
3. Execute a aplicação.
4. Utilize as rotas e endpoints disponíveis conforme documentação.
